# REACH Summer Build

This is a Google project. We build it with Google products, starting with the Antigravity IDE.

Here are the few things to do before we meet. We will set up the code and your tasks together at kickoff, so this is all you need for now.

## 1. Install the Antigravity IDE

Download and install the **[Google Antigravity IDE](https://antigravity.google/)**. Get the Antigravity *IDE*, the coding environment, not anything else by that name.

## 2. Send me your Google email

Reply to your welcome email with the **Google (Gmail) address** you want to use for the project. I use it to add you to our Google Cloud credits and turn on **Gemini** inside your Antigravity IDE. Gemini is your default assistant for the summer.

## 3. Add your own agent (optional)

Gemini is your default. If you also have a **Claude** or **ChatGPT** subscription and want an agent you already know:

1. In the Antigravity IDE, open the **Extensions** panel.
2. Search for your assistant and install it (for example, **Claude Code** for Claude, or the **Codex / ChatGPT** extension for ChatGPT).
3. Run it and **sign in with your existing subscription** when the extension prompts you to authenticate.

Don't have either one? You can skip this step. Step 2 already set you up with Gemini through our Google credits.

## 4. Join our Discord

This is where we talk day to day. The invite link is in your welcome email. Join before Tuesday.
